# Online Reservation System (Java + MySQL)

A desktop application developed using Java Swing, MySQL, and JDBC.

## Features
- User Sign Up
- User Login
- Ticket Reservation
- Ticket Cancellation
- Modern UI using FlatLaf
- MySQL Database Integration

## Technologies Used
- Java 17
- Swing GUI
- JDBC
- MySQL
- FlatLaf

## How to Run
1. Install Java 17
2. Install MySQL
3. Create database 'reservation_system'
4. Import tables from /src/db
5. Add JDBC & FlatLaf JARs from /lib
6. Run LoginForm.java
